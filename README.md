# Gestión de Datos de Países 🌍

Trabajo Práctico Integrador — Programación 1  
Tecnicatura Universitaria en Programación — UTN

## Descripción

Aplicación de consola en Python que permite gestionar un dataset de países,
aplicando listas, diccionarios, funciones, ordenamientos y estadísticas.
Los datos se almacenan y leen desde un archivo CSV.

## Estructura del proyecto

```
tpi_paises/
├── main.py           # Menú principal
├── datos.py          # Carga CSV, agregar y actualizar países
├── busqueda.py       # Búsqueda y filtros
├── ordenamiento.py   # Ordenamiento por criterios
├── estadisticas.py   # Estadísticas del dataset
├── utils.py          # Funciones de visualización compartidas
├── paises.csv        # Dataset base
└── README.md
```

## Requisitos

- Python 3.x
- No requiere librerías externas

## Cómo ejecutar

```bash
python main.py
```

## Funcionalidades

- Agregar y actualizar países
- Buscar por nombre (parcial o exacta)
- Filtrar por continente, rango de población y rango de superficie
- Ordenar por nombre, población o superficie (asc/desc)
- Estadísticas: máximos, mínimos, promedios, cantidad por continente
